# SpellingGenius

A spelling test app for kids, built in Swift.

My daughter needed spelling practice and the apps on the App Store were either ad-ridden or overcomplicated. So I built one.

## Features

- Create custom word lists for each week's spelling homework
- Text-to-speech reads out words for the child to spell
- Tracks progress and highlights words that need more practice

## Tech

- **Language:** Swift
- **Platform:** iOS
- **Testing:** Unit tests and UI tests included
- **Architecture:** Standard SwiftUI project
